### 🚀 功能亮点

只需点击几次鼠标就可以生成一份精美的 npm 包对比表格。

可以直接复制到 `Markdown` 文档或者 `HTML` 文档。

让你的技术调研报告看着 👍**高** 👍**大** 👍**上**
